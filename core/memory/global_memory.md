# Global Memory (Godot Harness)

## 通用技术基线

- 引擎优先基线：Godot 4.x（默认按 4.6 规范）。
- 语言优先基线：GDScript 严格类型、Godot 4 Callable 信号连接。
- 运行验证优先通过 MCP，避免纯静态推测。

## 通用工作流

1. 需求澄清 -> 影响范围识别 -> 方案确认。
2. 先更新文档索引与测试计划，再进行实现。
3. 实现后执行 smoke；关键路径执行 regression。
4. 回写开发日志与测试证据，保障跨会话接力。

## 通用风险清单

- 仅改代码不留测试证据。
- 同步逻辑无冲突仲裁与失败回滚。
- 过度依赖本地环境，导致跨设备不可复现。
